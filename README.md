# 🏋️ FitGenie AI

An AI-powered Fitness and Nutrition Web Application that provides personalized workout plans, meal recommendations, progress tracking, injury management, AI coaching, and premium subscription features.

---

## 🚀 Live Demo

🔗 Netlify: https://fitgenie-ai.netlify.app

---

## 📌 Features

- 🔐 Google Authentication
- 🍎 Apple Authentication
- 🤖 AI Workout Planner
- 🥗 AI Nutrition Planner
- 📈 Progress Tracking Dashboard
- 💬 AI Coach Chat
- 🛒 Grocery Planner
- 🩹 Injury Recovery Guidance
- 💎 Premium Subscription (Genie Pro & Genie Plus)
- 📊 Weekly Reports
- 👤 User Profile Management

---

## 🛠 Tech Stack

### Frontend
- HTML5
- CSS3
- JavaScript (ES6)

### Backend

- Firebase Firestore

## ⚙️ How to Run

Clone the repository:

```bash
git clone https://github.com/dhruv-jangid/fitgenie-ai.git
```

Run a local server:

```bash
python3 -m http.server 8080
```

Open:

```
http://localhost:8080
```
